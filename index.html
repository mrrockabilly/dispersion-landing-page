Dispersion

Serverless Web Hosting

Reduce Server Costs

We have made it simple to store data using IPFS, a peer-to-peer hypermedia protocol. This reduces the cost spent on running servers and makes web content more reliable.

Desktop Application

A desktop application that allows users to add files to their local IPFS node and publish them to a fixed address. This address can then be mapped to a human readable domain name.

github.com/dispersionJS/dispersion

IPFSmon

A command line to that automatically hashes development work. This allows developers to make changes to their work and quickly have those changes accessible online.
